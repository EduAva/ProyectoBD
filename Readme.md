##Base de Datos##
#Instrucciones para crear la base de datos#
1. Ingresar a postgres.
2. Ejecutar el siguiente comado para importar la base con el archivo adjunto:
    `\i ruta\del\archivo`

##Servidor##
#Instrucciones para levantar el servidor#
1. Tener un servidor instalado (PHP, XAMPP, WAMP, etc.) en este caso se uso PHP
2. Mover los archivos a un directorio.
3. Ejectuar la siguinte instruccion en el directorio:
`php -S localhost:8080`
